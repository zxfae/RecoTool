# PassiveTool 
Passive Tool is a recognition program written in Rust.

## Use Rustup to install Rust
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Usage
```bash
cargo run -- -i <ip address>
```
